# CampoSync - Sitio Web Principal

Página web principal del dominio camposync.com.mx

## Qué es este proyecto
Landing page / sitio web de CampoSync, una plataforma de operaciones de campo offline-first para empresas en Latinoamérica.

## Stack
- HTML, CSS, JS (sitio estático)

## Propósito
- Presentar CampoSync a clientes potenciales
- Explicar el producto y sus beneficios
- Captar leads / contacto de empresas interesadas
